## 학습순서
```
1. 플랫폼 선정: programmers 혹은 solved.ac(baekjoon online judge)
2. 언어 설정: c++ / java / python / javascript
3. 기본 문법 학습: 입출력 / 문자열 / 제어문 / 자료형
4. 자료구조 및 알고리즘 라이브러리: 배열 / 연결리스트 / 스택 / 큐 / 덱 / 셋 / 해시(맵) / 트리 / 그래프 / 정렬 / 수학
5. 문제풀이 및 분석: 문제유형 / 풀이과정
```
